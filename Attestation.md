# Attestation

Declaro que trabajé en este proyecto de manera individual. Utilicé a Claude Sonnet 4.5 como principal apoyo de inteligencia artificial, especialmente para tareas del backend. También hice uso de recursos externos como el servicio https://picsum.photos/ para la generación de imágenes y la librería de componentes HeroUI para acelerar el desarrollo.

Para el Frontend me demoré aproximadamente 2 horas en tener un prototipo completamente funcional con datos mock.

Las 3 horas restantes fueron para el backend, en el cual me demoré 2 horas y media entre la creación, integración al front y corrección de errores al generar el build de docker.
